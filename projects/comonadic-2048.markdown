description: A terminal implementation of 2048 in Haskell using the Store comonad to model the game grid. The comonadic approach lets the core movement and merging logic be expressed in just a few functions by using extend and experiment to operate over grid context. Includes memoization via data-memocombinators to keep the Store comonad performant.
